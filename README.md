# DecodeLabs UI/UX Project 2

## Project Overview
This project focuses on creating a low-fidelity wireframe for an e-commerce homepage.

## Objective
Improve user flow and information hierarchy while maintaining a clean layout.

## Features
- Navigation Bar
- Hero Section
- Categories Section
- Featured Products
- Why Choose Us
- Special Offer Banner
- Footer

## Tools Used
- Figma

## Design Constraints
- Low-Fidelity Wireframe
- Black, White and Grey colors only
- Focus on User Experience and User Flow
